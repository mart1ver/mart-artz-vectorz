SCRIPTS DISPONIBLES
===================

demo_scripts/defile_formes.py
  Show complet ~5 minutes : intro (20s) + défilé 15 formes (6s chacune) + finale 5 actes (180s).
  La finale utilise 24 spots simultanés (max univers DMX 0).
  Usage :
    python3 demo_scripts/defile_formes.py                  # 15 formes × 6s + finale 180s
    python3 demo_scripts/defile_formes.py 127.0.0.1 4     # 15 formes × 4s + finale 180s

demo_scripts/artnet_text.py
  Animation typographique multi-scènes avec triangle/cœur en arrière-plan.
  Cycle : ArtNet (4s) → controled (4s) → generator (4s) → made by: (4s) → Martin VERT (6s)
  Effets : couleurs HSV animées, chromatic aberration, sobel, pixelate strobe (scène 5).
  Usage :
    python3 demo_scripts/artnet_text.py

demo_scripts/lettre_a.py
  Affiche un seul caractère 'a' noir sur fond blanc. Script de test/référence.
  Usage :
    python3 demo_scripts/lettre_a.py

demo_scripts/system_validation.py
  Validation de la connexion ArtNet et des performances système.
  Usage :
    python3 demo_scripts/system_validation.py


PRÉREQUIS
=========
1. Lancer LuxCore dans Processing (martz_artz_verctorz.pde)
2. Vérifier que le nombre de spots est configuré dans le GUI Processing

ARCHITECTURE TECHNIQUE
======================
Protocole      : ArtNet UDP port 6454
Cible défaut   : 127.0.0.1 (localhost)
Univers        : 0

Structure DMX (28 paramètres de base) :
  Canal  1- 3  : RGB background
  Canal  4-19  : 4 blades (A/B/C/D) — 2 canaux 16-bit chacun
  Canal 20     : Blend mode
  Canal 21-22  : Blur A (size) et B (sigma)
  Canal 23     : Pixelate
  Canal 24     : Sobel (bistable)
  Canal 25     : RGB Split
  Canal 26-27  : Saturation / Vibrance
  Canal 28     : Chromatic aberration (bistable)

Structure par spot (20 canaux, offset = 28 + spot_id × 20) :
  +0 +1 +2    : RGB fill
  +3          : Alpha
  +4          : Stroke weight
  +5          : Stroke alpha
  +6 +7 +8    : RGB stroke
  +9 +10      : Taille Pan  (16-bit MSB+LSB)
  +11 +12     : Taille Tilt (16-bit MSB+LSB)
  +13 +14     : Rotation    (16-bit MSB+LSB)
  +15 +16     : Position Pan  (16-bit MSB+LSB)
  +17 +18     : Position Tilt (16-bit MSB+LSB)
  +19         : Mode (forme 0-14)

Capacité max   : 256 spots
Performance    : 44-49 FPS avec 50 spots
